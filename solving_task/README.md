# Marta: карта заводов РФ

Статический сайт с интерактивной картой заводов и промышленных площадок РФ.

## Что внутри

- `index.html` - страница приложения.
- `styles.css` - стили интерфейса.
- `app.js` - логика карты, фильтров, списка и карточек.
- `data/factories.json` - выгрузка объектов из OpenStreetMap.
- `scripts/update_data.py` - скрипт обновления базы.

## Запуск

Из папки `Marta` запустите локальный сервер:

```powershell
python -m http.server 8080
```

Потом откройте в браузере: `http://localhost:8080`.

## Обновление базы заводов

```powershell
python scripts/update_data.py
```

Скрипт перезапишет `data/factories.json` актуальными объектами OSM.

## Примечание

База зависит от наполненности OpenStreetMap: чем лучше размечены объекты в OSM, тем полнее карта.
